# BABEL TOWER SYSTEM ENGINE

**UNFILTERED SOUND SYSTEM**  
A retro pixel-style web audio experiment.

## Overview

A browser-based sound engine inspired by the Tower of Babel. Built with strong retro Apple II + BBS aesthetics.

## Features

- Pixel art interface with scanline effect
- UNFILTERED sound generation: Random Sine, Square, Sawtooth, and Noise
- **シーケンサー** (8-step Sequencer)
- HOLD MODE for sustained tones
- Babel Chorus (Chaos Mode) - Multiple overlapping voices
- Visual Tower Building System - Stack layers to grow the tower
- Tower Collapse button = Full System Reset

## Controls

| Button              | Function                        |
|---------------------|---------------------------------|
| 波形再生             | Play random waveform            |
| 混沌モード            | Trigger Babel Chorus            |
| シーケンサー          | Toggle 8-step Sequencer         |
| HOLD MODE           | Toggle sustained sound          |
| ▲ 塔を高くする ▲      | Increase tower height           |
| 塔を崩す             | Reset entire system             |
